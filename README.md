# Fictitious Corporate World Satire

A static satire website built with HTML, CSS, JavaScript, and PNG cartoons.

## Local preview

Open `index.html` in your browser.

## GitHub Pages deployment

1. Create a GitHub repository named `fictitious-corporate-world-satire`.
2. Push this folder to the repository.
3. In the repo settings, enable GitHub Pages from the `main` branch and root folder.
4. If using a custom domain, add a file named `CNAME` containing your custom domain.
5. Update your DNS:
   - Add a CNAME record pointing your custom domain to `jmaret.github.io`.

## Files

- `index.html` — main page
- `style.css` — site styles
- `script.js` — front-end animation
- `images/` — generated cartoon PNG assets
